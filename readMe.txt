第一个入门mabatis的例子

实现步骤：
    1.新建student表
    2.加入maven的mybatis坐标和mysql驱动的坐标
    3.创建实体类---Student，用于保存表中的一行数据
    4.创建持久层的dao接口，定义操作数据库的方法
    5.创建mabatis使用的配置文件，也叫sql映射文件：用于写sql语句，是一个xml类型的文件，一般一个表对应一个sql映射文件
        ①这个xml文件应放在与接口同级的目录中
        ②文件名与接口名一致
    6.创建mybatis的主配置文件：一个项目就一个主配置文件，这个文件提供了数据库的连接信息和sql映射文件的位置
    7.创建使用mybatis类，通过mybatis访问数据库
